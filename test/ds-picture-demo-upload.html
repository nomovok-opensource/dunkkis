File Upload

This script is for uploading images into the Dunkkis test database for testing purposes. There are no error checks associated with the uploading, so be careful with your input.

1 )Choose a file to upload.

2) Enter a logtime for the file (yyyy-mm-dd hh:mm:ss).
If you enter no logtime, present time will be used.

3) Enter a sensor id. (eg. "26.C0295C000000")

4) Enter a device id. (eg. "0E469C00")
